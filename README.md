# 爱翻译隐私政策

最后更新日期：2026 年 5 月 23 日

爱翻译是一款网页翻译浏览器扩展，用于帮助用户翻译网页正文、选中文字、动态弹出菜单以及图片或视频画面中的可见文字。

本扩展不会出售、出租或转让用户数据，不会将用户数据用于广告、画像、信用评估或贷款目的。

## 本扩展可能处理的数据

**网站内容：**当用户主动使用翻译功能时，扩展会读取当前网页的可见文字、选中文字，或视觉截图中的文字内容，并将这些内容发送到用户自行配置的 AI 翻译接口，以获得翻译结果。

**个人身份信息：**扩展不会主动收集个人身份信息。但如果用户翻译的网页内容中包含姓名、邮箱、地址等信息，这些内容可能会随翻译请求一起发送到用户配置的 AI 接口。该处理仅用于完成用户主动触发的翻译功能。

**设置数据：**扩展会在本地保存用户的语言偏好、模型配置、悬浮窗开关状态、翻译缓存和加密后的 API Key。这些数据用于保持用户设置、提升翻译速度和减少重复请求。

## API Key 安全

用户填写的 API Key 会加密后保存在浏览器本地存储中。扩展不会向作者或其他第三方服务器上传 API Key。

## 第三方服务

扩展本身不提供翻译服务器。用户可以自行配置 DeepSeek、OpenAI、Claude、Xiaomi MiMo 或其他兼容接口。翻译请求会发送到用户配置的接口服务商，相关数据处理也会受到对应服务商隐私政策约束。

## 数据用途

扩展处理数据仅用于提供网页翻译、选中文字翻译、视觉翻译、连续翻译、翻译缓存和用户设置保存功能。扩展不会将数据用于与核心翻译功能无关的用途。

## 数据共享

扩展不会向作者出售、出租或转让用户数据。除用户自行配置的 AI 翻译接口外，扩展不会主动向其他第三方共享用户数据。

## 用户控制

- 用户可以在扩展设置中修改或删除 AI 接口配置。
- 用户可以关闭自动翻译、视觉连续翻译和页面悬浮窗。
- 用户可以通过浏览器扩展管理页面卸载扩展并删除本地保存的数据。
